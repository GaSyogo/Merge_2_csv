添付の
tset.csv
について，以下の処理を行なってください．

1. U列からAV列には ns_co_*, ns_nc_*
というラベルの列が並んでいますが，* が同じ2つの列を1つにまとめる．
たとえば U列の見出しは ns_co_C3 ですが，
AK列に co と nc しか違わない，ns_nc_C3 があります．
これを ns_C3 という新しい列を作ってまとめてください．
つまり ns_C3 には，ns_co_C3 と ns_nc_C3 の和を入れてください．
たとえば U10=0.6, AK10=0.25 なので，新しい列には 0.85 が入ります．

2. AWからGWについても同じことを行なってください．
ここには ec_co_*, ec_ex_*, ec_in_* というラベルの列が並んでいますが，
* が同じ3つの列を1つにまとめること．新しい列の名前は ec_* でお願いします．
同じように和をとってください．
*が同じ列が2つしかない場合も，1つにまとめてください．
つまり ec_co_XX, ec_in_XX は存在するが，ec_ex_XX が存在しない場合でも，
2つまとめて ec_XX を作ってください．
